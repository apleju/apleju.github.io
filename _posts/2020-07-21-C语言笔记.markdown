### 二、C程序初识

#### 1.c程序基本构成
- 程序的组成（程序主体、编译预处理、注释）
    - 程序主体：函数定义是C程序的主体，程序的功能由函数来完成
    - 编译预处理：每个以符号#开头的行，称为编译预处理行。C编译系统提供了大量的内部函数，这些函数放在（.h）库中，而把函数的说明放在标题文件，也称头文件中，通过#include命令完成，其一般形式为：#include<文件名>或#include "文件名"
    - 注释：注释是用/*...*/括起的内容，是一种提示或记号

- 函数的组成
    - 每个函数（包括主函数）的定义分为两部分：函数首部和函数体
    - 函数首部，即函数的第一行，包括函数名、函数类型、参数类型、形参名。其一般形式为：函数类型 函数名（参数类型 形参名1，参数类型，形参名2...）,例如：int Max(int x,int y)
    - 函数体由一对大括号{}括起的语句序列，包括变量说明和实现函数功能的语句两部分。
                实现函数功能的语句可以是C语言语句，也可以是库函数或自己设计的函数调用语句。

- 语句的组成
    - 每个语句最后用“；”结尾，分号是语句的组成部分，函数说明语句后不能有分号。
    - 语句中含有各种关键字、运算符、表达式等。
    - 可以在一行写多句，也可以一句写多行

- 程序结构的特点
    - 由一个或多个函数构成，但有且只有一个主函数。可以有多个子函数，并在主函数中调用。
    - 用main()开始执行，也在main()函数中结束。main()函数可以放在任意位置。
    - C语言中提供了丰富的函数，称为库函数，若要调用库函数，在调用前必须将被包含文件嵌入源文件中。
    - 变量必须先定义再使用，未定义的变量不能使用。
    - 主函数可以调用任何子函数，子函数之间也能互相调用，但子函数不能调用主函数。

### 三、C语言的语法

#### 1.字符集
- 字符是C语言最基本的元素，C语言源程序中不能出现字符集以外的字符
    - 英文字母：A-Z,a-z（52个）
    - 数字：0-9（10个）
    - 键盘符号：33个
    - 空白符

#### 2.关键字
- 关键字是一类有特定含义的单词，用来说明数据类型、存储类型、访问说明、语句等。列入关键字表的单词一律不得移作他用，用户定义的标识符不得与它们冲突，因此又称为保留字。

#### 3.标识符
- 指用来表示变量名、函数名、数组名、类型名、文件名等的有效字符序列，C语言中没有标准标识符的概念，main可以看作唯一的标准标识符，为编译程序预定义的主函数的名字，ANSIC规定标识符的命名必须满足以下规则：
    - 第一个字符必须是字母或下划线
    - 标识符其他部分必须由字母、下划线、数字组成
    - 区分字母大小写
    - 一般环境允许标识符长度为32个字符，但标识符不宜太长
    - 不能与关键字重名

#### 4.常量
- 固定不变的量，包括整型常量、浮点常量、字符常量、字符串常量等

#### 5.运算符
- 用来进行某些操作的单词，当运算符作用于被操作对象后，将获得一个结果，常见的有赋值运算符、算术运算符、关系运算符、逻辑运算符

#### 6.分隔符
- 用来定界或分隔其他语法成分的单词，常用的分隔符有逗号、分号、空格、回车换行符等

- 基本类型

- 长整型

#### 2.实型常量
- 具有一定精度，又称为浮点数

#### 3.字符常量
- 用单引号括起来的一个字符，字母区分大小写
- 以反斜杠（\）引导的字符序列，被称为转义字符

#### 4.字符串常量
- 用双引号括起来定界一串字符

#### 5.符号常量
- 用一个标识符来代表一个常量。用#define定义；用conse定义

### 三、变量
#### 1.变量及说明
- 其值可以被改变的量称为变量

#### 2.整型变量
- 基本整型int

#### 3.实型变量

### 四、运算符和表达式
- 运算符按功能分为：算术运算符、关系运算符、逻辑运算符、逗号运算符、位运算符、赋值运算符
- 表达式：用运算符把操作数连接起来所构成的句子，操作数可以是常量、变量、函数

#### 1.算术运算符和算术表达式
- 算术运算符：+、-、*、/、%
- 自增、自减运算符：++k,k++(--k,k--)
- 算术表达式：用算术运算符和括号将运算对象连接起来，按照运算符优先级别，从高到低运算，同级则自左至右运算

#### 2.关系运算符和关系表达式
- 关系运算符：>  >=  <  <=  ==  !=
- 关系表达式：表达式 < 关系运算符 > 表达式

#### 3.逻辑运算符和逻辑表达式
- 逻辑运算符：“&&  //  ! ”  
- 逻辑表达式：< 运算符 > 表达式 或 表达式 < 运算符 > 表达式

#### 4.赋值运算符和赋值表达式
- 赋值运算符：“ = ”     
- 赋值表达式： 变量 = 表达式

#### 5.其他运算符及表达式
- 逗号运算符和表达式
    - 运算符：,（将若干个表达式连接起来）
    - 表达式：表达式1，表达式2，表达式3，....表达式N

- 条件运算符和条件表达式
    - 运算符：？  ：
    - 表达式：e1 ？ e2 : e3;
#### 6.数据类型转换
- 混合运算时数据类型的自动转换。将低类型向高类型转换，俗称“就长不就短”

- 赋值运算时数据类型的自动转换。将赋值运算符右边表达式的数据先转换成左边变量的数据类型，再赋值，“就左不就右”

- 数据类型的强制转换。利用强制类型转换运算符将一个表达式的数据类型强制的转换成指定数据类型。一般形式为：（类型名）（表达式）。例如（double）a、（int）（x+y）

### 五、数据的输入/输出
#### 1.字符数据的输入/输出
- 输入函数： getchar(),一般形式为：c= getchar()

- 输出函数： putchar(),一般形式为： putchar(c);

#### 2.格式输入/输出
- 输出函数：printf("格式控制字符串"，输出列表);

- 输入函数：scanf("格式控制字符串"，地址表列);

## 第三章
### 一、顺序结构
#### 1.C语言的语句
- 表达式语句。表达式后加分号构成表达式语句
            
- 复合语句。用大括号括起来的若干语句

- 控制语句。用于控制程序的流程，实现分支和循环结构。由系统特定的关键词加条件等语法成分组成。有9种控制语句，分为以下3类：
    - 构成分支结构的条件判断控制语句：if语句、switch语句
    - 循环控制结构的语句：do-while语句、while语句、for语句
    - 控制流程的转移语句：break语句、continue语句、return语句、goto语句

- 函数调用语句。通过调用一个函数实现其功能。一般形式为：函数名（实际参数表）；

- 空语句

#### 2.顺序结构程序设计
- 按照语句在程序中的先后顺序依次执行的结构。主要由简单语句、复合语句、输入/输出语句构成。

### 二、分支结构
#### 1.  if(if-else)语句

- if 结构（单分支结构）。
    - 一般形式为：

            if(表达式){
                语句;
            }

- if-else 结构（双分支结构）
    -  一般形式为：

            if(表达式){
                语句;
            }else{
                语句;
            }

- if-else-if语句（多分支结构）
    - 一般形式为：

            if(表达式){
                语句;
            }else if (表达式){
                语句;
            }

- if 语句的嵌套
    - 一般形式为：

            if(表达式){
                if(表达式){
                    语句；
                }else{
                    语句；
                }
            }else{
                语句;
            }

#### 2.switch 语句
- 一般形式为：

            switch(表达式){
                case 常量表达式 1 :
                    语句组;
                    break;
                case 常量表达式 2 :
                    语句组;
                    break;
                    .
                    .
                    .
                case 常量表达式 n :
                    语句组;
                    break;
                default:语句组n+1;
            }

### 三、循环结构
#### 1.while语句
- 一般形式为：

            while（表达式）{
                语句;
            }

#### 2.do-while 语句
- 一般形式为：

            do{
                语句;
            }while(表达式);

#### 3.for 语句
- 一般形式为：

            for(表达式1;表达式2;表达式3){
                语句;
            }

#### 4.转移语句
- break语句，中断语句
    - 功能：强制中断当前的循环结构，直接结束循环。
    - 其一般形式为：

            break;
 
- continue语句，结束本次循环，开始下次循环。
    - 功能：结束本次循环，即不再执行循环体中continue之后的语句，转入下一次循环条件的判断与执行。
    - 一般形式为：

            continue;

#### 5.循环的嵌套。
- 一个循环结构的循环体中包含另一个循环结构，称为循环的嵌套或多重循环。
- 外面的循环语句称为“外层循环”，循环体包含着的循环称为“内层循环”。
- 3种循环语句可以互相嵌套。
