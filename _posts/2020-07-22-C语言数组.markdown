# C语言
## 第四章 数组
### 一、数组的概念
- 数组是将具有同一属性的数据放在一起的集合
- 有数组名，数组元素。
- 数组元素由其所在位置序号（下标）标识
    - 按照数组元素类型分为：数值数组、字符数组、指针数组、结构数组等
    - 按照数组元素下标个数不同，分为：一维数组、二维数组、多维数组等
### 二、一维数组
#### 1.定义
- 一般形式为：类型说明符 数组名[常量表达式]
    - 数组名命名规则与变量名命名规则相同，不能重复定义
    - 常量表达式表示数组中元素个数，下标从0开始
    - 常量表达式需为确定数值，可以是数值、字符或符号常量，但不能为变量

#### 2.一维数组元素引用
- 数组名[下标]
- 下标通常为整型常量或整型表达式
#### 3.一维数组的初始化
- 一般形式为：类型说明符 数组名[常量表达式]={初值表};
- 对数组的所有元素赋初值：数据放在{}内，用逗号隔开
    - 必须逐个赋值
    - 说明数组时可以不指定长度，系统根据大括号内数据的个数，自动定义长度
- 对数组的部分元素赋初值
    - 按顺序赋值，没有数据按0自动赋值
